== Description ==
Products Variation Limit is an easy an intuive way to increase woocommerce products variations limit.

Unzip and Upload the directory ‘products-variation-limit’ to the ‘/wp-content/plugins/’ directory

Activate the plugin through the ‘Plugins’ menu in WordPress

Go to Wocommerce > Variation Limit menu and set variation limits. Default limit is 100.

== Installation ==
1) Unzip and Upload the directory ‘products-variation-limit’ to the ‘/wp-content/plugins/’ directory

2) Activate the plugin through the ‘Plugins’ menu in WordPress

3) Go to Wocommerce > Variation Limit menu and set variation limits. Default limit is 100.
